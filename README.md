<h1>Active Directory - Bulk User Creation</h1>



<h2>Description</h2>
This project is a simple PowerShell script that creates active directory user accounts from a source file. This script was designed to save time and prevent manual user entry errors. The PowerShell script will populate an OU and implement the appropriate group policies, based on the source file user data.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>AD</b>

<h2>Environments Used </h2>

- <b>Windows 11</b> (22H2)
